## Quick Start

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd flight-analyzer-ui
```

2. Install dependencies:

```bash
npm install
```

### Development Server

```bash
npm start
```

The application will be available at `http://localhost:4200/`

### Production Build

```bash
npm run build
```
